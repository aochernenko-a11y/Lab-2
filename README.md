# Лабораторна робота №2  
## Тема: Інкапсуляція та властивості у мові C#

### Мета роботи
Ознайомитись із принципами **інкапсуляції** в об’єктно-орієнтованому програмуванні та навчитися створювати класи з використанням **властивостей** (`get`, `set`, автозаповнені, обчислювальні, з різними рівнями доступу).

---

### Склад проєкту
У проєкті **Lab-2** реалізовано кілька класів:

| Клас | Призначення |
|------|--------------|
| **`Product`** | Моделює товар. Містить приватні поля, властивості з валідацією, автозаповнену властивість `Category`, обчислювальну властивість та властивість з обмеженим `set`. |
| **`BoardGame`** | Приклад об’єкта настільної гри. Має властивості для назви, жанру, кількості гравців, віку, ціни, а також обчислювальну властивість `IsPartyGame`. |
| **`GameGenre`** | Перелічення типів жанрів (enum), використовується у класі `BoardGame`. |
| **`Program`** | Точка входу програми. Містить меню, створює об’єкти класів, демонструє зміну властивостей та обробку винятків. |

---

### Основні можливості
- Робота з **приватними полями** та **публічними властивостями**.  
- Використання **автовластивостей** із початковими значеннями.  
- Реалізація **обчислювальних властивостей** без зберігання проміжних полів.  
- Демонстрація властивостей із різними рівнями доступу (`get` / `private set`).  
- Валідація даних через **приватні допоміжні методи**.  
- **Обробка винятків** (`try-catch`) під час зміни властивостей.  

---

### Використані концепції ООП
- **Інкапсуляція** — обмеження доступу до полів через `private`.  
- **Абстракція** — виділення суттєвих характеристик об’єктів (товар, гра).  
- **Безпечна взаємодія** — зміна стану об’єкта лише через властивості з перевіркою даних.  

---

### Як запустити
1. Відкрити проєкт `Lab-2.sln` у **Visual Studio**.  
2. Обрати конфігурацію **Debug** → **Any CPU**.  
3. Натиснути **F5** або **Start** для запуску.  
4. У вікні консолі виконати команди меню для тестування класів.

### Висновок
У результаті виконання лабораторної роботи створено програму, яка демонструє практичне застосування принципів **інкапсуляції** та **властивостей** у C#.  
Робота відповідає вимогам завдання і містить приклади з автовластивостями, обчисленнями, рівнями доступу та обробкою помилок.
